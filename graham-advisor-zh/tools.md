# 格雷厄姆分析工具

## 工具1：安全边际计算表

```
┌─────────────────────────────────────────────────┐
│               安全边际计算表                      │
├─────────────────────────────────────────────────┤
│ 公司名称：_________________________               │
│ 内在价值（保守估计）：___________               │
│ 当前价格：_______________________               │
│                                                  │
│ 安全边际 = (内在价值 - 价格) / 内在价值         │
│           = (________ - ________) / ________     │
│           = __________ %                        │
│                                                  │
│ 格雷厄姆标准：安全边际 > 30% 才考虑买入          │
│ 激进标准：安全边际 > 50% 更安全                 │
└─────────────────────────────────────────────────┘

判断：
□ 安全边际 > 50% — 强烈买入信号
□ 安全边际 30-50% — 可以考虑
□ 安全边际 < 30% — 谨慎，等待更好价格
□ 安全边际 < 0 — 价格高于价值，不买
```

## 工具2：内在价值估算表

### 方法1：格雷厄姆公式

```
V = EPS × (8.5 + 2g)

其中：
- V = 内在价值
- EPS = 每股收益（TTM或预期）
- 8.5 = 经验系数（代表合理的P/E基准）
- g = 未来7-10年预期年增长率（%）

EPS = ___________
g (预期增长率) = ___________ %
V = ___________ × (8.5 + 2 × ___________)
V = ___________

当前价格：___________
安全边际 = (_________ - _________) / _________ = _______%
```

### 方法2：清算价值法

```
总资产 = ___________
总负债 = ___________
清算价值 = 总资产 - 总负债 = ___________

注意：清算价值假设公司立即清盘，
      适合有形资产重的公司（工业、金融）
      不适合轻资产公司（科技、服务）

每股清算价值 = ___________ / 股份数 = ___________
当前股价 = ___________
比率 = 当前股价 / 每股清算价值 = ___________

格雷厄姆标准：比率 < 1/3 才有安全边际
```

### 方法3：现金流折现法（简化版）

```

步骤1：计算未来5年现金流现值
年份1：________ / (1+____%)^1 = ________
年份2：________ / (1+____%)^2 = ________
年份3：________ / (1+____%)^3 = ________
年份4：________ / (1+____%)^4 = ________

小计 = ___________

永续价值 = 年金现值系数 × 最后一年现金流
         = (1 / (折现率 - 永续增长率)) × 最后一年现金流

企业价值 = 5年现金流现值 + 永续价值现值
         = ________ + ________ / (1+____%)^5
```

## 使用说明

这些工具是为了让你把格雷厄姆的思维框架**实际应用**。

使用方法：
1. 遇到投资决策时，拿出安全边际计算表
2. 用内在价值估算表估算企业价值
3. 用投资决策清单做最后检查
4. 用财务健康检查表快速筛选
5. 用市场情绪温度计判断买入时机

记住：**格雷厄姆的核心是保守**。宁可错过机会，也不要在不确定的情况下冒险。
